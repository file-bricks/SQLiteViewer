# Store Listing - SQLiteViewer

## Deutsch

### Kurzbeschreibung (max 100 Zeichen)
Leichtgewichtiger SQLite-Browser - Datenbanken öffnen, durchsuchen und lokal abfragen.

### Beschreibung (max 10.000 Zeichen)
SQLite Viewer Pro ist ein schneller, portabler SQLite-Datenbank-Browser. Er öffnet `.db`-, `.sqlite`- und `.sqlite3`-Dateien sofort und ermöglicht das Durchsuchen, Abfragen und Exportieren von Daten - ohne SQL-Kenntnisse und ohne externe Abhängigkeiten.

**Was SQLite Viewer Pro kann:**

- Tabellen-Browser: Listet automatisch alle Tabellen auf und zeigt Daten in einem sortierbaren Raster an
- Schema-Ansicht: Zeigt `CREATE TABLE`-Definitionen mit Syntax-Highlighting
- SQL-Editor: Führt eigene SQL-Abfragen mit Syntax-Highlighting und Ergebnisansicht aus
- Volltextsuche: Echtzeit-Suche über alle sichtbaren Spalten
- CSV-Export: Tabellen oder Abfrageergebnisse als CSV-Datei exportieren
- JSON-Export: Aktuelle Ansicht optional als `sqliteviewer-export-v1.json` für Companion-Workflows exportieren
- Sortierung: Spaltenköpfe anklicken für auf- oder absteigende Sortierung
- Tastenkürzel: `Ctrl+O`, `Ctrl+F`, `Ctrl+E`, `F5`, `F9`

**Warum SQLite Viewer Pro statt DB Browser oder DBeaver?**

- Sofortiger Start: Keine lange Ladezeit, kein Java, kein Framework
- Portabel: Eine einzige Python-Datei, läuft überall dort, wo Python installiert ist
- Leichtgewichtig: ~750 Zeilen Code, minimaler Speicherverbrauch
- Keine Installation nötig: Starten und sofort nutzen
- Read-only-Zugriff: SQLite-Dateien werden bewusst nicht versehentlich verändert

**Für wen ist SQLite Viewer Pro?**

Entwickler, Datenanalysten und Power-User, die schnell einen Blick in SQLite-Datenbanken werfen möchten. Ideal als schlanker Ersatz für schwergewichtige Datenbank-Tools bei einfachen Analyse-, Prüf- und Exportaufgaben.

## English

### Short Description (max 100 chars)
Lightweight SQLite browser - open, search, and query databases locally in seconds.

### Description (max 10,000 chars)
SQLite Viewer Pro is a fast, portable SQLite database browser. It opens `.db`, `.sqlite`, and `.sqlite3` files instantly and lets you browse, query, and export data without SQL knowledge and without external dependencies.

**What SQLite Viewer Pro does:**

- Table Browser: Automatically lists all tables and displays data in a sortable grid
- Schema View: Inspect `CREATE TABLE` definitions with syntax highlighting
- SQL Editor: Write and execute custom SQL queries with syntax highlighting and result view
- Full-Text Search: Real-time search across all visible columns
- CSV Export: Export any table or query result to a CSV file
- JSON Export: Optionally export the current visible view as `sqliteviewer-export-v1.json`
- Sorting: Click column headers to sort ascending or descending
- Keyboard Shortcuts: `Ctrl+O`, `Ctrl+F`, `Ctrl+E`, `F5`, `F9`
- Read-only Access: Open SQLite files safely without accidental edits

**Why SQLite Viewer Pro over DB Browser or DBeaver?**

- Instant startup: No long loading times, no Java, no framework overhead
- Portable: A single Python file, runs anywhere Python is installed
- Lightweight: ~750 lines of code, minimal memory footprint
- No installation required: Launch and use immediately
- Focused workflow: Browse, inspect, and export without the overhead of a full database suite

**Who is SQLite Viewer Pro for?**

Developers, data analysts, and power users who need a quick look into SQLite databases. Ideal as a slim replacement for heavyweight database tools in straightforward review, debugging, and export workflows.
